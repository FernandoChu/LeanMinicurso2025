# Formalizando Matemáticas en Lean

Este es el repositorio para el minicurso *Formalizando Matemáticas en Lean* dado en el IMCA en Dicembre, 2025.
Aquí se puede encontrar el material del curso, así como los ejercicios y presentaciones.


## Instalación

Para seguir este curso, es necesario instalar Lean. Las instrucciones de instalación se encuentran [aquí](https://docs.lean-lang.org/lean4/doc/quickstart.html).

Luego de instalar Lean, para utilizar el material de este curso haga click en el símbolo de "para todo" y seleccione como se indica en la siguiente imagen.
`Open Project > Project: Download Project`:
![download project dialog](images/vscode-download-project.png)

Aparecerá una caja para ingresar texto, y ahí pegue el link de este repositiorio
```
https://github.com/FernandoChu/LeanMinicurso2025
```

Tras seleccionar un lugar de instalación (No seleccione en Onedrive!), el repositorio se descargará e instalará en su computadora. Esto podrá durar unos minutos. Nota: Posiblemente el instalador preguntará si desean instalar Git y Elan, instale ambos programas.

El material del minicurso se puede encontrar en el folder `Lean4MiniCurso`.
